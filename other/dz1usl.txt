+ 1. Установить себе git.
 Пользователям Windows могу порекомендовать msysgit https://msysgit.github.io
+ 2. Зарегистрироваться на сайте GitHub, создать там свой первый репозиторий (не забудьте инициализировать его сразу,
поставив галочку "Создать файл README"!)
+ 3. Испробовать установленный себе git в действии, склонировав свой свежесозданный репозиторий себе на компьютер.
+ 4. Внести какие-нибудь изменения, убедиться, что у вас работают команды git add, git commit, git push,
разобраться - как именно они работают.
+ 5. Взять себе тот код, который мы писали на уроке и продолжить его:

+ 5.1. Создать страницу с формой добавления новости в базу и обработчик этой формы, который новость добавит.
+ 5.2. Создать страницу, которая показывает одну новость. ID этой новости передается методом GET.
+ 5.3. Создать страницу, которая отображает форму редактирования новости. ID формы для редактирования также передается методом GET.
 Записывать в базу отредактированную новость пока не нужно, достаточно формы!
